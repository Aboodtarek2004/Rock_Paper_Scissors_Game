# Rock Paper Scissors Game

A simple Rock Paper Scissors game where the player competes against the computer. The player chooses rock, paper, or scissors, the computer makes a random choice, and the game decides the winner using the classic rules.

## Game Rules

- Rock beats scissors
- Scissors beats paper
- Paper beats rock
- If both choices are the same, the round is a draw

## Features

- Player choice selection
- Random computer choice
- Round result display
- Beginner-friendly game logic
- Easy to expand with score tracking, reset controls, animations, or sound effects

## How To Play

1. Choose rock, paper, or scissors.
2. The computer randomly chooses one option.
3. The winner is displayed based on the game rules.
4. Play again and try to beat the computer.
