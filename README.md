# 🎮 Stone Paper Scissors Game

A fun and beginner-friendly **Stone Paper Scissors** game built using **C++**. Challenge the computer, play **3 exciting rounds**, and see who wins! 🏆

---

# 📖 Description

This project is a console-based implementation of the classic **Stone 🪨 Paper 📄 Scissors ✂️** game.

The player competes against the computer. Each round, both choose one of the three options, and the game determines the winner according to the classic rules. After **3 rounds**, the program announces the overall winner and gives the player the option to play again. 🔄

---

# ✨ Features

* 🎮 Player vs Computer gameplay
* 🎲 Random computer choices
* 📊 Displays the result of every round
* 🏆 Calculates the final winner
* 🔄 Play again without restarting the program
* 🧩 Clean code using **Enums**, **Structs**, and **Functions**
* 💻 Simple console interface

---

# 🎯 Game Rules

| Player Choice | Beats       |
| ------------- | ----------- |
| 🪨 Stone      | ✂️ Scissors |
| 📄 Paper      | 🪨 Stone    |
| ✂️ Scissors   | 📄 Paper    |

If both players choose the same option, the round ends in a **🤝 Draw**.

# 🚀 How to Play

1. ▶️ Run the program.
2. 🎮 Enter your choice:

   * **1** → 🪨 Stone
   * **2** → 📄 Paper
   * **3** → ✂️ Scissors
3. 🤖 The computer makes a random choice.
4. 🏅 The winner of the round is displayed.
5. 🔁 After 3 rounds, the overall winner is announced.
6. 🎉 Choose whether to play again.
